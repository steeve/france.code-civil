Article L654-7
----
I. - Les personnes morales peuvent être déclarées responsables pénalement, dans
les conditions prévues par l'article 121-2 du code pénal, des infractions
prévues par les articles L. 654-3 et L. 654-4.

II. - Les peines encourues par les personnes morales sont :

1° L'amende, suivant les modalités prévues par l'article 131-38 du code pénal ;

2° Les peines mentionnées à l'article 131-39 du code pénal.

III. - L'interdiction mentionnée au 2° de l'article 131-39 du code pénal porte
sur l'activité dans l'exercice ou à l'occasion de l'exercice de laquelle
l'infraction a été commise.
