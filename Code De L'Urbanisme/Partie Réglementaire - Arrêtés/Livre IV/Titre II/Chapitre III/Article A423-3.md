Article A423-3
----
La structure du numéro d'enregistrement de treize chiffres est la suivante :

Le numéro de code géographique INSEE du département (trois caractères) ;

Le numéro de code géographique INSEE de la commune (trois caractères) ;

Les deux derniers chiffres du millésime de l'année de dépôt de la demande (deux
caractères) ;

Le numéro de dossier composé de cinq caractères ; le premier (de ces cinq
caractères) est réservé au service instructeur ; les quatre autres (caractères)
sont utilisés pour une numérotation en continu par nature d'autorisation ou acte
relatif à l'utilisation du sol.
