Article R233-60
----
Dans les stations de sports d'hiver et d'alpinisme classées, mentionnées à
l'article R. 143-33, l'établissement de la taxe de séjour est autorisé après une
enquête à laquelle il est procédé dans les formes établies par les articles R.
11-3 à R. 11-13 du code de l'expropriation pour cause d'utilité publique.
