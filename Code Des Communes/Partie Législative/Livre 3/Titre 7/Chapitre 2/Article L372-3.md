Article L372-3
----
Conformément à l'article 112 du code rural, le déversement dans un cours d'eau
domanial d'eaux usées provenant d'égouts communaux est autorisé par un acte
déclarant d'utilité publique les travaux [**]conditions de forme[**] ; cet acte
détermine les conditions auxquelles le déversement est subordonné en vue de
sauvegarder les intérêts généraux.
