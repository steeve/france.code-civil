Article L233-43
----
Un règlement d'administration publique détermine les formalités à remplir par
les logeurs, hôteliers, propriétaires ou autres intermédiaires chargés de
percevoir la taxe de séjour.

Le même règlement d'administration publique fixe les pénalités pour infractions
à ces dispositions.

Les pénalités ne peuvent dépasser le triple du droit [*somme due*] dont la
commune a été privée [*proportion*].
