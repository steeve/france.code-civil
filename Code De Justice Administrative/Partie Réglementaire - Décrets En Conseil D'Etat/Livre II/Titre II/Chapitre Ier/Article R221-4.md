Article R221-4
----
Les tribunaux administratifs désignés ci-après sont composés de plusieurs
chambres, dont le nombre est fixé comme suit :

Amiens : 3 chambres ;

Bastia : 2 chambres ;

Besançon : 2 chambres ;

Bordeaux : 4 chambres ;

Caen : 2 chambres ;

Cergy-Pontoise : 5 chambres ;

Châlons-en-Champagne : 2 chambres ;

Clermont-Ferrand : 2 chambres ;

Dijon : 3 chambres ;

Grenoble : 5 chambres ;

Lille : 6 chambres ;

Limoges : 2 chambres ;

Lyon : 7 chambres ;

Marseille : 8 chambres ;

Melun : 5 chambres ;

Montpellier : 5 chambres ;

Nancy : 2 chambres ;

Nantes : 4 chambres ;

Nice : 6 chambres ;

Orléans : 3 chambres ;

Pau : 2 chambres ;

Poitiers : 3 chambres ;

Rennes : 5 chambres ;

Rouen : 3 chambres ;

Strasbourg : 4 chambres ;

Toulouse : 4 chambres ;

Versailles : 7 chambres.
