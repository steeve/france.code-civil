Article R561-3
----
L'enquête est menée dans les formes prévues par les articles R. 11-4 à R. 11-14
du code de l'expropriation pour cause d'utilité publique.

Le dossier mentionné à l'article R. 561-2 du présent code est adressé également
par le préfet, pour avis, à chaque commune dont une partie du territoire est
comprise dans le périmètre délimitant les immeubles à exproprier.

L'avis du conseil municipal doit être transmis au préfet dans un délai de deux
mois. Passé ce délai, l'avis est réputé favorable.
