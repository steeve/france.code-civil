Article R6152-218
----
L'avancement d'échelon s'effectue suivant les durées suivantes :

1er échelon : un an ;

2e échelon : un an ;

3e échelon : deux ans ;

4e échelon : deux ans ;

5e échelon : deux ans ;

6e échelon : deux ans ;

7e échelon : deux ans ;

8e échelon : deux ans ;

9e échelon : deux ans ;

10e échelon : deux ans ;

11e échelon : deux ans ;

12e échelon : quatre ans.

L'avancement d'échelon est prononcé par le préfet.
