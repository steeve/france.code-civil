Article R206
----
La référence à l'article 27 (deuxième alinéa) du code de l'administration
communale doit être remplacée, pour l'application du présent code à la
Nouvelle-Calédonie, par la référence à l'article L. 121-12 du code des communes
de la Nouvelle-Calédonie et, pour l'application du même code en Polynésie
française, par la référence à l'article L. 121-12 du code des communes
applicable localement.
