Article R221-52
----
Dans le cas prévu à l'article 31-1 du code civil, la demande est portée devant :

1° Le tribunal dans le ressort duquel est situé le domicile du demandeur, si
celui-ci réside en France ;

2° Le tribunal dans le ressort duquel est situé le lieu de naissance du
demandeur, si celui-ci est né en France et réside à l'étranger ;

3° Le tribunal d'instance du 1er arrondissement de Paris, si le demandeur est né
et réside à l'étranger.
