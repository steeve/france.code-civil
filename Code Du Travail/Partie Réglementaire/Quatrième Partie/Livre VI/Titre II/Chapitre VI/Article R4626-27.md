Article R4626-27
----
Bénéficient d'une surveillance médicale renforcée : 1° Les femmes enceintes ; 2°
Les mères d'un enfant âgé de moins de deux ans ; 3° Les jeunes travailleurs âgés
de moins de dix-huit ans ; 4° Les travailleurs handicapés ; 5° Les agents
réintégrés après un congé de longue durée ou de longue maladie ; 6° Les agents
ayant changé de type d'activité ou d'établissement pendant une période de
dix-huit mois à compter de leur nouvelle affectation.
