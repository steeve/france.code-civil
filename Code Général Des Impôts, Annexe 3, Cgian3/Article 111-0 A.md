Article 111-0 A
----
Sans préjudice des dispositions de l'article 502 du code général des impôts, les
niveaux mentionnés au 3° du I de l'article 302 G dudit code sont égaux ou
supérieurs à :

a) 10 litres pour les alcools et les boissons spiritueuses ;

b) 20 litres pour les produits intermédiaires ;

c) 90 litres pour les vins et autres produits fermentés, dont 60 litres pour les
vins mousseux ;

d) 60 litres pour les vins mousseux ;

e) 110 litres pour les bières.
