Article 46 AJ
----
Constituent des organismes de formation agréés par l'Etat pour l'application du
deuxième alinéa de l'article 199 quater E du code général des impôts :

1° Les chambres de commerce et d'industrie ;

2° Les chambres de métiers ;

3° Les centres de gestion agréés prévus à l'article 1649 quater C du code
général des impôts, pour les actions de formation de leurs adhérents non
couvertes par la cotisation d'adhésion ;

4° Les fonds d'assurance-formation prévus à l'article L. 961-10 du code du
travail ;

5° Les organismes de formation qui ont été préalablement agréés à cet effet par
le ministre chargé du commerce et de l'artisanat.
