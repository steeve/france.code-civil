Article 50-0 bis
----
Le montant de la fraction de taxe d'apprentissage obligatoirement réservée au
développement de l'apprentissage en application de l'article L. 118-3 du code du
travail est fixé conformément aux dispositions de l'article D. 118-7 du code
précité pour la métropole et aux dispositions du 5° de l'article D. 811 du même
code pour les départements d'outre-mer.
