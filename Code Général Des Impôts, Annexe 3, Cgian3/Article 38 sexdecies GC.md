Article 38 sexdecies GC
----
La valeur comptable des stocks apportés dans les conditions mentionnées au
sixième alinéa du I de l'article 151 octies du code général des impôts par un
exploitant qui a opté pour les dispositions du I de l'article 72 B du code
général des impôts est reprise pour la même valeur si cet exploitant n'opte pas
pour les dispositions du III du même article. Elle est reprise pour la même
valeur dans les inventaires suivants dans la mesure où les stocks apportés n'ont
pas encore été vendus.
