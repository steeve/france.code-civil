Article R958-22
----
Toute demande d'autorisation comporte les informations suivantes : 1° Nom du
navire ; 2° Numéro et port d'immatriculation ; 3° Marques extérieures
d'identification ; 4° Nom et adresse du propriétaire ou de l'affréteur ; 5°
Tonnage brut ; 6° Longueur hors tout ; 7° Puissance du moteur ou des moteurs ;
8° Signal distinctif ; 9° Fréquences radios utilisées ; 10° Méthode de pêche ;
11° Espèces qu'il est prévu de capturer ; 12° Période de pêche pour laquelle
l'autorisation est sollicitée.
